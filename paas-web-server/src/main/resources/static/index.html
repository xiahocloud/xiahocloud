🚀 Spring GraphQL Demo

欢迎使用 Spring GraphQL 演示应用！
这个应用展示了如何使用 Spring Boot 和 GraphQL 构建现代化的 API。

🔧 开发工具

GraphiQL 查询界面 H2 数据库控制台

📚 GraphQL 端点

GraphQL API 端点: /graphql

示例查询：

query { users { id name email age posts { id title content } } }

示例变更：

mutation { createUser(input: { name: "新用户" email: "newuser@example.com" age: 28 }) { id name email age } }

🛠️ 功能特性

  • 用户管理 (CRUD 操作)
  • 文章管理 (CRUD 操作)
  • GraphQL 查询和变更
  • 关联数据查询
  • 异常处理
  • 数据验证
提示： 点击 "GraphiQL 查询界面" 可以直接在浏览器中测试 GraphQL 查询和变更操作。
